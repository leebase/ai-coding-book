- **Format**: Mermaid (` ```mermaid `) for diagrams that benefit from flowcharts/sequence diagrams. ASCII art for simple checklists or structures that work better inline.

- **Consistency**: Same color/style conventions across all Mermaid diagrams. Use flowchart TD (top-down) as default direction.

### Format Options

| Format | Pros | Cons |
|--------|------|------|
| Markdown + companion GitHub repo | Simple, version-controlled, free | No traditional distribution |
| PDF (via pandoc/typst) | Professional, portable, printable | Needs typography pass |
| EPUB | E-reader friendly | Complex markdown features may not render |
| Static site (mdBook, Docusaurus) | Interactive, searchable, linkable | Not a "book" in traditional sense |
| Leanpub | Built-in audience, markdown-native | Platform lock-in |

**Recommendation**: Markdown repo as source of truth → pandoc/typst for PDF → consider Leanpub or Gumroad for distribution.
