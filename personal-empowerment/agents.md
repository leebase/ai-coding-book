# Personal Empowerment — Agent and Skill Registry

This file records the active agent and skill stack for the Personal Empowerment book.
It exists so the method can be resumed consistently after session breaks.

Treat this as an active workflow file, not as passive reference.
If you start a session in `personal-empowerment/`, read this file as part of
startup.

## Activation Order

When working in this directory:

1. Load `skills/thought-leadership-book.md`
2. Load `skills/personal-empowerment-book.md`
3. Use this registry to choose the right agent or review layer
4. Load `skills/sounds-like-lee.md` for any chapter draft revision, cadence
   pass, or final polish

The business-growth workspace already operated with an explicit reviewer and
supporting-skill stack. This workspace should use the same level of activation
discipline, but with the personal-handbook stack that matches this book.

## Required Process Rule

- Any chapter-level drafting or revision should run through
  `skills/sounds-like-lee.md` before it is treated as finished prose.
- One-sentence paragraphs are allowed only for deliberate emphasis.
- Before marking a chapter pass complete, run the `Cadence Gate` in
  `skills/sounds-like-lee.md` and fix any consecutive one-sentence paragraphs
  unless they are clearly intentional hinges.
- If the active stack changes, update this file in the same session.
