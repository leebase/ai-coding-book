# Skill: Thought Leadership Writing

Use this skill when drafting chapters, proposals, or long-form argument for a
thesis-driven nonfiction book.

## Writing Pattern

1. Open with tension, not exposition
2. Name the wrong but tempting move
3. Show the consequence
4. Reframe the problem
5. Give the reader a more useful model
6. End with a decision, implication, or forward pull

## Paragraph Rules

- One point per paragraph
- Move from claim to evidence to implication
- Prefer concrete language over abstract summary
- Cut any sentence that only sounds smart

## Use Story Carefully

- Start with a scene when it sharpens the argument
- Do not let the anecdote outrun the thesis
- After the story, say what it means

## Use Evidence Carefully

- Quote or paraphrase only what advances the argument
- Distinguish fact, interpretation, and forecast
- If the evidence is thin, narrow the claim

## Tone Discipline

- Write as a person, not a firm
- Prefer lived recognition to prestige signaling
- If a sentence sounds like it came from a keynote slide, cut it
- Keep the prose human enough that a reader can feel judgment, invitation, and
  reflection on the page
- If the passage makes the reader feel managed rather than understood, rewrite it
